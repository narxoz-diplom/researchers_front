# 06. Course Detail Page

## Цель
Страница курса: описание + список уроков + пэйволл для подписчиков без подписки.

## Маршрут
`/courses/:id`

## Источник данных
- `GET /api/v1/courses/:id` → `CourseDetailDto { ..., lessons: LessonSummary[], hasAccess }`.
- `GET /api/v1/me/subscription` (опционально, чтобы знать срок).

## Макет

- **Hero-блок:**
  - Слева: обложка `aspect-video rounded-2xl`.
  - Справа: заголовок, имя автора, описание, мета (количество уроков, дата создания), CTA:
    - `hasAccess=true` → кнопка «Перейти к первому уроку».
    - `hasAccess=false` → бейдж «Требуется подписка» + кнопка «Получить доступ» (открывает модалку с инструкцией обратиться к админу).
- **Раздел уроков:**
  - Список нумерованных уроков (`<LessonRow />`):
    - порядковый номер, заголовок, длительность видео (если есть).
    - иконка `Lock` для запертых, `CheckCircle` для пройденных.
  - При клике:
    - если `hasAccess` — переход в `/courses/:id/lessons/:lessonId`;
    - иначе — открыть пэйволл-модалку.
- **Если автор/админ:** показать кнопку «Редактировать курс» (ссылка в Studio).

## Состояния
- Loading, error, not found, archived (предупредительный бейдж только для админа/автора).

## Хуки
- `useCourseQuery(id)`
- `useMySubscription()`
- `useCourseProgressQuery(id)` (см. `progress`)

## Definition of Done
- [ ] Подписчик без подписки видит пэйволл и не может открыть урок.
- [ ] Автор своего курса видит контент без подписки и видит кнопку редактирования.
- [ ] Прогресс отображается рядом со списком уроков.
- [ ] Адаптив для мобильных (Hero колонки -> в одну).
